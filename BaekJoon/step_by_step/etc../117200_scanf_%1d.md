# Scanf를 공백없이 입력받기
-------------------------------------------------------------------------------------

### scanf("%1d",&N)

만약, 54321을 입력했을 때, 54321은 입력버퍼 stdin으로 가서 임시 저장된 후 입력이 끝났다는 신호가 오면(\n, ,\t)

 ----stdin----
 
|             |

|   54321\n   |

|             |

 -------------
 
 
차례대로 %1d의 수만큼 N변수에 저장

(만약, %2d면 N = 54 --> N = 32 --> N = 1\n ,이런식!)


```c
#include<cstdio>
#include<cstdlib>


int main()
{
	int N;
	int hap = 0;
	scanf("%d", &N);

	for (int i = 0; i < N; i++)
	{
		int x = 0;
		scanf("%1d", &x);
		hap += x;
	}

	printf("%d", hap);
	
	system("pause");
	return 0;
}
```
