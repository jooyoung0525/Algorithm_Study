# 입력받은 문자열을 10개씩 끊어 출력하는 문제!!!!

-----------------------------------------------------------------------------------------

**완전 유용한 팁**

1. while(scanf("%d",&N) == 1)) //scanf의 리턴값(아마 %d가 리턴인자..? 같음)이 1일때 무한 반복!
2. %1d ==> 정수 한개만 입력받음
3. %10s ==> 문자열 10개 이하만 입력받음


-------------------------------------------------------------------------------------------

```c
#include<cstdio>
#include<cstdlib>


int main()
{
	char N[100];
	

	while (scanf("%10s", &N) == 1)
		printf("%s\n", N);	
		
	
	system("pause");
	return 0;
}
```
